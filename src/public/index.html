How It Works

ขั้นตอนการ Login

แอปย่อยส่ง user มาที่ SSO — ผ่าน Google แล้วได้ JWT Token กลับไป

1
แอปย่อย Redirect
ส่ง app_id + redirect_uri มาที่ /login
2
Google OAuth
Login ด้วยบัญชี @northbkk.ac.th
3
ตรวจสอบสิทธิ์
ดึงสิทธิ์ 2 แกน (Role + Scope) จาก IAM
4
ออก JWT Token
เซ็น RS256 อายุ 8 ชั่วโมง
5
Redirect กลับ
ส่ง ?token=eyJ... ไปยังแอปย่อย
